# Abdul Azeez M — Portfolio

A neo-brutalist personal portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no dependencies — just raw code with bold design choices.

This is a personal project created to experiment with UI design, animations, and interactions. It focuses more on creativity and learning rather than strict production-level practices.

---

## What is Neo-Brutalism?

Neo-brutalism (also called neo-brutalist web design) is a UI style inspired by brutalist architecture — raw, bold, and unapologetically loud. It avoids overly polished designs and instead emphasizes strong visual structure and contrast.

* **Thick black borders** — strong, visible edges
* **Flat offset shadows** — no blur, just solid layered depth
* **High-contrast colors** — bold and vibrant palettes
* **Visible layout structure** — nothing hidden behind smooth effects
* **Playful imperfections** — scrapbook-style elements and hand-crafted feel

This portfolio follows a **scrapbook-style neo-brutalist approach**, using torn paper effects, animated highlights, and interactive UI elements.

---

## Design Highlights

* **Paper Tear Sections** — SVG-based torn edges with scroll effects
* **Scroll Highlights** — text highlights animate like marker strokes
* **Matrix Typing Effect** — animated hero text
* **Decorative Animations** — floating and falling UI elements
* **Dark/Light Theme Toggle**
